# 实体宠物生产链路优化 A/B（2026-07-14）

## 生产结论

1. 实体版默认使用 `adaptive_green_temporal_v3`，光流分辨率固定为 `384`。
2. 萌宠版保持原链路，不启用实体毛发时序修正。
3. 实体版 Seedream 提示词禁止轮廓光，并使用“无地面单色合成底板”约束。
4. Seedream 仍可能生成绿幕地面和接触阴影，生产链路必须继续执行 `normalize_bg()`；不能只靠提示词替代背景归一化。
5. 回退方式：`PETAVATAR_GREEN_TEMPORAL_REFINE=0` 回到逐帧 `edge_v2`。

## 时序抠图 A/B

测试素材为两只实体猫的完整 `fast_walk`、`sleep` 动作，每段 97 帧。v3 使用双向 Farneback 光流、光度门控 alpha 融合、光流对齐碎片投票和大部件保护。

| 宠物 | 动作 | 时序误差 v2 → v3 | 碎片率 v2 → v3 | CPU 总耗时 v2 → v3 |
|---|---|---:|---:|---:|
| 银灰短毛猫 | 快走 | 0.019712 → 0.018190（-7.7%） | 0.003170% → 0.000732%（-76.9%） | 30.60s → 43.00s |
| 银灰短毛猫 | 睡眠 | 0.001247 → 0.000752（-39.7%） | 0.000024% → 0（-100%） | 29.68s → 58.63s |
| 虎斑白袜猫 | 快走 | 0.025455 → 0.023890（-6.1%） | 0.010173% → 0.006040%（-40.6%） | 28.13s → 44.39s |
| 虎斑白袜猫 | 睡眠 | 0.001824 → 0.001124（-38.4%） | 0.000016% → 0（-100%） | 27.74s → 46.87s |

静态指标变化可控：背景 alpha 泄漏绝对增量约 `0.00012–0.00017`，前景损失绝对增量约 `0.00008–0.00025`；目视未发现拖影、尾巴被削或主体闪回。

### 384 分辨率选择

快走素材上，384 相比 480 的时序误差只差约 0.5%，但光流 pass 平均从约 19.5 秒降至约 15.6 秒，约快 20%；320 更快但碎片和时序指标继续回退。因此生产取 384。
