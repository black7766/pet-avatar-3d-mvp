## 方案脉络

1. 先用 Seedream 把用户宠物照片标准化为实体版和萌宠版。
2. 用 Seedream 生成快走和睡眠目标状态首帧。
3. 用 Seedance 生成 4 秒无声目标状态循环视频并锁定同状态首尾帧。
4. 本地一次解码、并行自适应绿幕抠图；实体版用 12px 自然毛发边缘 profile，萌宠版用保守 8px profile，并保护高饱和绿色身份细节。
5. 合成透明 WebP/WebM。
6. 页面通过状态按钮和历史卡片展示原图、实体版、萌宠版。
7. 当前动作资产只保留 `idle/fast_walk/sleep`；都以稳定循环资产为主，过渡动画不是当前默认规格。
