开心 / 进食高帧动作对比

同一组 6 张动作关键帧，这次直接拉到 45fps、60fps、超采样 60fps 三档。这里不调用任何云端模型，重点验证“继续大幅补帧”是否还能明显提升体感。

素材加载中 Canvas 实时播放 不触发下载

对比口径

增强帧和接近 60fps 两档保持接近原动作时长；超采样 60fps 会播放全部中间帧，所以动作更慢，但能看出当前本地光流插帧的细腻度上限。

成本与时间

本地插帧API 成本 0；当前电脑单动作三档通常数秒内完成，适合 MVP 批量打磨。
重生关键帧若改用 image_gen/Seedream 直接生成更多原生姿态，按每个动作一次或多次生图请求计费，质量上限更高但一致性需要筛选。
视频方案Seedance/Kling 类图生视频最自然，但需要视频 API 费用，并且还要裁帧、抠透明、压 WebP/APNG。
