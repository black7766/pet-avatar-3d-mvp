# 历史实验清理记录

清理日期：2026-07-15

## 目的

项目以 `airgap_motion_v4` 为唯一生产链路。历史模型对比、A/B 页面和中间帧不再保留在工作目录，避免旧方案被误用，并降低本地占用。

## 已删除

本次共删除 176 个目录或文件集合，约 48,195.53 MB：

- `.venvs/`：8 套 matting 模型虚拟环境。
- `.models/`：BiRefNet、BEN2、SAM2、MatAnyone、PaddleMatting 等测试权重和缓存。
- `matting_bench/data/`、`matting_bench/outputs/` 和各 provider 的运行产物。
- 旧 WebP MVP：`legacy_webp_mvp/`、`poc_output/mvp_webp/`。
- 2026-07-10 至 2026-07-15 的生成端、时序、照明和结构增强 A/B 输出。
- `_inspect_*`、`_debug_*`、`_ab_*`、benchmark 页面资源和历史落盘帧。
- 旧 A/B 报告脚本 `ab_compare_report.py` 及对应报告。

模型 provider 的源码、安装说明、评测结论和 V4 回归测试仍保留。以后需要复验模型时，可以按 provider README 重建环境和重新下载权重。

## 已保留

- `poc.py`、`prompts.py`、`petavatar_server.py` 和全部生产测试。
- 所有正式历史宠物目录及其透明 WebP、状态首帧、raw 付费视频和指标。
- 所有 `inputs/` 原图。
- 当前 V4 样例 `pet_20260715_155012_95feab9d_real`。
- 主对比页面 `poc_output/paimomo_compare.html`。
